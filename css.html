O que é CSS?

CSS (Cascading Style Sheets)

CSS é usado para estilizar o conteúdo das páginas web, como cores, fontes e layouts.

h1 { color: blue; }

Exemplo de código:

body {
  font-family: Arial, sans-serif;
}

h1 {
  color: red;
}
