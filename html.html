HTML: A Estrutura da Web

HTML (HyperText Markup Language) não é uma linguagem de programação, mas sim uma linguagem de marcação. Ele atua como o esqueleto de um site, definindo sua estrutura e organização.

Usando tags como `

` (título), `

` (parágrafo) e `` (imagem), você informa ao navegador como o conteúdo deve ser exibido. Sem o HTML, não haveria conteúdo para ser estilizado ou manipulado. É a primeira linguagem que todo desenvolvedor web aprende.

Exemplo de Código HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Minha Página</title>
  </head>
  <body>
    <h1>Olá, Mundo!</h1>
    <p>Este é um parágrafo.</p>
  </body>
</html>

O que cada tag faz no exemplo acima:

< !DOCTYPE html> e < html>: Indicam que o documento é um HTML5.
< head>: Contém metadados sobre o documento, como o conjunto de caracteres e o título da página. O conteúdo dentro do não é visível na página.
< body>: Contém todo o conteúdo visível da página.
< header>: Define o cabeçalho da página, incluindo o título e a navegação (

).
< nav>: Agrupa os links de navegação do site.
< main>: Contém o conteúdo principal e único da página.
< section>: Usada para agrupar conteúdo tematicamente relacionado. No exemplo, é a primeira seção do conteúdo principal.
< article>: Representa um conteúdo autossuficiente e independente, como um post de blog.
< aside>: Contém conteúdo secundário que está relacionado ao conteúdo principal, como uma barra lateral.
< footer>: Define o rodapé da página, que geralmente inclui informações de direitos autorais e links de contato.
