HTML5 Descrição

HTML5 é a quinta e mais recente versão do HyperText Markup Language, a linguagem de marcação usada para estruturar e apresentar conteúdo na web. É a espinha dorsal de qualquer página da internet, e o que a torna tão especial é a quantidade de novos recursos e melhorias que ela trouxe em relação às versões anteriores.

novidade sobre

Novas tags semânticas: Em vez de usar apenas < div > para tudo, o HTML5 introduziu tags como < header> , < nav >, < article >, < section > e < footer >. Isso não apenas torna o código mais organizado e fácil de ler, mas também ajuda os motores de busca e as tecnologias assistivas (como leitores de tela) a entenderem a estrutura da página.

O que o HTML5 faz?

O HTML5 é a mais recente versão da Linguagem de Marcação de Hipertexto (em inglês, HyperText Markup Language), que é o código que usamos para estruturar o conteúdo de uma página da web. É a espinha dorsal de qualquer site que você visita.

Pricipais tags do html5

Tags Estruturais e Semânticas Essas tags ajudam a estruturar o conteúdo da página de forma lógica, facilitando a leitura para navegadores, mecanismos de busca e pessoas que usam tecnologias assistivas. < header>: Define a parte introdutória de uma seção ou página, geralmente contendo o título, um logotipo ou a navegação principal. < nav>: Usada para agrupar links de navegação. < main>: Marca o conteúdo principal e único de uma página. Só pode haver uma tag < main> por documento. < article>: Representa uma parte do conteúdo que pode ser distribuída ou reutilizada de forma independente, como uma postagem de blog, um artigo de jornal ou um comentário. < section>: Agrupa conteúdo relacionado. É usada para dividir o conteúdo de um < article> ou de uma página em seções lógicas. < aside>: Define um conteúdo que está relacionado ao conteúdo principal, mas é separado dele (como uma barra lateral, por exemplo). < footer >: Representa o rodapé de uma seção ou página, geralmente contendo informações de direitos autorais, links para a política de privacidade ou dados de contato.

Tags Multimídia

Essas tags substituíram a necessidade de plugins como o Flash para incorporar áudio e vídeo em páginas web. < audio>: Incorpora conteúdo de áudio, como músicas ou podcasts. < video>: Incorpora conteúdo de vídeo. Tags de Formulário e Interatividade Elas aprimoram a forma como os usuários interagem com os formulários. < input>: A tag já existia, mas o HTML5 introduziu novos tipos para ela, como: type="email": Valida se o texto digitado é um endereço de e-mail. type="url": Valida se o texto é uma URL. type="date": Exibe um seletor de data. type="search": Usado para campos de busca. < figure> e < figcaption>: Usadas para agrupar e legendar imagens, gráficos ou código. < figure> é o contêiner e < figcaption> é a legenda. Tags Gráficas Essas tags permitem a criação de gráficos e animações diretamente no navegador. < canvas>: Cria uma "tela" onde você pode desenhar gráficos e animações usando JavaScript. < svg>: Usada para desenhar gráficos vetoriais escaláveis. É ideal para logotipos, ícones e ilustrações que não perdem a qualidade ao serem redimensionados. Essas são algumas das tags mais importantes introduzidas no HTML5. Elas foram criadas para tornar o código mais semântico (com significado) e para dar mais recursos nativos para os desenvolvedores, tornando a criação de páginas web mais eficiente e moderna.
