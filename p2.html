AGORA VAMOS PRA CSS

PRA QUE SERVER??

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata) e é a linguagem que usamos para estilizar páginas da web. Enquanto o HTML cria a estrutura e o conteúdo de uma página (como títulos, parágrafos e imagens), o CSS cuida da aparência. Com o CSS, você pode fazer coisas como: Mudar as cores de textos e fundos. Ajustar o tamanho e o tipo das fontes. Posicionar elementos na tela. Adicionar animações e efeitos visuais. Pense no HTML como o esqueleto de uma casa e o CSS como a decoração, a pintura das paredes e a escolha dos móveis. Você pode ter o esqueleto, mas é o CSS que torna a casa bonita e agradável de se olhar. Para usar o CSS, você cria regras que dizem ao navegador como um elemento específico deve ser exibido. Por exemplo, uma regra pode dizer que todos os títulos principais da sua página devem ser azuis e centralizados.
